# Little Lemon Restaurant Website

A responsive restaurant website for **Little Lemon**, a family-owned Mediterranean restaurant in Chicago. This project showcases seasonal menus, special offers, table booking, and opening hours, with a modern and user-friendly design.

## Live Preview

Check out the live website here: [Little Lemon Live](https://littlelemon-1-57ni.onrender.com/home/)

## Features

- Home page with special offers and featured menu items  
- About section with restaurant story and owners  
- Dynamic menu display (Italian, Greek, and Turkish cuisine)  
- Table reservation functionality  
- Opening hours section  
- Responsive design for mobile and desktop  

## Tech Stack

- **Backend:** Django  
- **Frontend:** HTML, CSS, JavaScript
